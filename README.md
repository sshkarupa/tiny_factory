# TinyFactory

Rebuilding FactoryBot with basic functionality

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tiny_factory'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tiny_factory
